# DAVIVIENDA-Reto
Reto de OrangeHRM ( Automatización, API Automatización, Pruebas performance )

// Ejecucion E2E Por medio de KATALON

1.Descargar desde HUB el repositorio  por medio de [https://github.com/Aledassilva/DAVIVIENDA-Reto/tree/main/DAVIVIENDA ](https://github.com/Aledassilva/DAVIVIENDA-Reto.git) .ZIP 
2.Abrir Project DAVIVIENDA-Reto desde Katalon 
3.Abrir Rama Test Cases > DAVIVIENDA OrangeHRM (Doble Clik) 
4.Parte superior derecha Tap en (>) Run para correr caso de prueba de Automatizacion E2E 
5.Debe abrirse un navegador (Chrome) automaticamente y correr el caso de prueba Automatizado caso de prueba Exitoso

// Ejecucion APi's por medio de KATALON

1.Descargar desde HUB el repositorio [https://github.com/Aledassilva/Autom_Reto/](https://github.com/Aledassilva/DAVIVIENDA-Reto.git)  por medio de .ZIP 
2.Abrir Project DAVIVIENDA-RETO desde Katalon 
3.Abrir Rama Test Cases > AUTOM Consultar usuarios o AUTOM Crear usuarios (Doble Clik) 
4.Parte superior derecha Tap en (>) Run para correr caso de prueba de Automatizacion API 
5.Al ejecutarse debe marcar los servicios exitosos en codigo (200) el cual paso los servicios Get, Post
6.Si desea cambiar las variables Abrir en Objet Repository > Postman > AUTOM Consultar usuarios o AUTOM Crear usuarios y seleccionar en el menu Variables, en esa seccion podra Cambiar las variables
